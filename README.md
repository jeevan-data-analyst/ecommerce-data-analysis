# 📊 E-commerce Product Mapping Analysis

## 📌 Overview
This project focuses on analyzing product mapping and data quality issues in eCommerce datasets. The goal is to identify gaps in product mapping and improve overall data consistency.

---

## 🎯 Objectives
- Analyze mapped vs unmapped products
- Identify brands with high "No Map" percentage
- Evaluate data completeness
- Detect data quality issues affecting mapping

---

## 🛠 Tools Used
- SQL (MySQL)
- Excel

---

## 📂 Dataset
The dataset contains product-level information including:
- Product Code
- Brand
- Title
- Size & Unit
- Map Status
- SKU Information

---

## 📊 Key Analysis Performed

### 1. Mapping Performance
- % of Mapped vs No Map products
- Overall mapping efficiency

### 2. Brand-Level Analysis
- Top brands with highest unmapped products
- % No Map by brand

### 3. Size Unit Analysis
- Mapping gaps by Size Unit (ounce, ml, etc.)

### 4. Data Quality Checks
- Missing brand values
- Duplicate records
- Inconsistent size units

---

## 📈 Key Insights
- Identified brands contributing most to unmapped products
- Found data inconsistencies impacting mapping accuracy
- Highlighted areas where data cleaning is required
