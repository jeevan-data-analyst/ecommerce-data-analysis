# ecommerce-data-analysis
SQL &amp; Power BI analysis of product mapping and data quality
## 📌 Project Overview
This project analyzes product mapping and data quality issues in eCommerce datasets.

## 🎯 Objectives
- Analyze mapped vs unmapped products
- Identify data quality issues
- Improve mapping efficiency

## 🛠 Tools Used
- SQL (MySQL)
- Excel

## 📊 Key Analysis
- % Mapped vs No Map
- No Map by Brand
- No Map by Size Unit
- Data Completeness Analysis

## 🔍 Key Insights
- Identified brands with highest mapping gaps
- Found data quality issues affecting mapping accuracy
